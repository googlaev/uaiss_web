# uaiss_web

## О проекте

Веб-приложение для управления экзаменами и статусами сотрудников. Позволяет:

- Отслеживать сроки действия экзаменов
- Управлять статусами (больничный, отпуск, командировка)
- Просматривать статистику сотрудников (администратору)

---

## Установка и запуск

### 1. Установить Docker Desktop

Скачать и установить Docker Desktop с официального сайта:

- **Windows/Mac:** https://www.docker.com/products/docker-desktop/

После установки **запустить Docker Desktop** и дождаться зеленого индикатора.

### 2. Скачать проект

Через Git:

```bash
git clone https://github.com/googlaev/uaiss_web.git
cd uaiss_web
```

### 3. Запустить приложение

```bash
docker compose up -d
```

### 4. Открыть в браузере

Перейти по адресу: http://localhost:8000

### 5. Войти в систему

| Роль | Email | Пароль |
|------|-------|--------|
| Сотрудник | user@uaiss.ru | `123456` |
| Администратор | admin@uaiss.ru | `admin123` |
